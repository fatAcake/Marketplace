**Users**

| Метод | Эндпоинт             | Описание                                   | Доступ         |
| ----- | -------------------- | ------------------------------------------ | -------------- |
| POST  | `/api/auth/register` | Регистрация нового пользователя            | Все            |
| POST  | `/api/auth/login`    | Авторизация пользователя                   | Все            |
| GET   | `/api/users/me`      | Получить информацию о текущем пользователе | Авторизованный |
| PUT   | `/api/users/me`      | Редактировать профиль                      | Авторизованный |
| GET   | `/api/users/:id`     | Получить информацию о другом пользователе  | Авторизованный |

**Products**

| Метод  | Эндпоинт                  | Описание                                | Доступ                   |
| ------ | ------------------------- | --------------------------------------- | ------------------------ |
| GET    | `/api/products`           | Список всех товаров (каталог)           | Все                      |
| GET    | `/api/products/:id`       | Детальная информация о товаре           | Все                      |
| POST   | `/api/products`           | Создать новый товар                     | Авторизованный           |
| PUT    | `/api/products/:id`       | Редактировать свой товар                | Авторизованный, владелец |
| DELETE | `/api/products/:id`       | Удалить свой товар                      | Авторизованный, владелец |
| GET    | `/api/users/:id/products` | Список товаров конкретного пользователя | Все                      |

**Orders**

| Метод | Эндпоинт                 | Описание                                       | Доступ                                                 |
| ----- | ------------------------ | ---------------------------------------------- | ------------------------------------------------------ |
| POST  | `/api/orders`            | Создать новый заказ с товарами (OrderItem)     | Авторизованный                                         |
| GET   | `/api/orders`            | Список своих заказов (покупатель)              | Авторизованный                                         |
| GET   | `/api/orders/:id`        | Детали конкретного заказа                      | Авторизованный, покупатель/продавец участвует в заказе |
| GET   | `/api/sales`             | Список заказов по своим товарам (для продавца) | Авторизованный, продавец                               |
| PUT   | `/api/orders/:id/status` | Изменить статус заказа (например, завершён)    | Авторизованный, владелец заказа                        |

**Discounts**

| Метод  | Эндпоинт                      | Описание                        | Доступ                          |
| ------ | ----------------------------- | ------------------------------- | ------------------------------- |
| POST   | `/api/discounts`              | Создать скидку для товара       | Авторизованный, владелец товара |
| GET    | `/api/discounts`              | Список всех скидок (по товарам) | Все                             |
| GET    | `/api/products/:id/discounts` | Скидки конкретного товара       | Все                             |
| PUT    | `/api/discounts/:id`          | Обновить скидку                 | Авторизованный, владелец скидки |
| DELETE | `/api/discounts/:id`          | Удалить скидку                  | Авторизованный, владелец скидки |

**PriceHistory**

| Метод | Эндпоинт                          | Описание                                        | Доступ                          |
| ----- | --------------------------------- | ----------------------------------------------- | ------------------------------- |
| GET   | `/api/products/:id/price-history` | Получить историю изменения цены и скидок товара | Авторизованный / все            |
| POST  | `/api/products/:id/price-history` | (опционально) добавить запись изменения цены    | Авторизованный, владелец товара |

**Additional**

| Метод | Эндпоинт                | Описание                            | Доступ                           |
| ----- | ----------------------- | ----------------------------------- | -------------------------------- |
| GET   | `/api/analytics/sales`  | Аналитика продаж по товарам, период | Авторизованный, продавец         |
| GET   | `/api/analytics/prices` | Аналитика цен и скидок, период      | Авторизованный / админ (или все) |
